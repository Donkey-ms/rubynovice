# Rubynovice

Ruby programming mastering platform using Github

## インストール法
- githubで自分のところにdaddygongon/rubynoviceをforkする
- cloneする

## 使用法
- docにある課題を解いていく．
- specをかける
- 自分の名前をつけたファイルを./lib/rubynovice/donkey.rbなどとして準備する．
- class Donkeyに解答をつくる
- 書き方はdonkey.rbを参照のこと．
